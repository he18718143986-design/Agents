# ADR-008：品牌命名、中文名与品牌哲学

> 状态：**部分采纳**（哲学已确认；中文名/slogan 待商标检索后终审） ｜ 2026-07-05

## 背景

商业化需要：ICP 备案的纯中文网站名称、可注册商标的品牌、与竞品形成认知区隔的品牌叙事。MIT 许可不含商标授权，不可使用 OpenHands 品牌。

## 决策

1. **品牌哲学（已确认）**：Stage 三重词源（舞台/阶段/逐步）+ "分阶段 = 人类的在场"作为核心价值主张，完整叙事见 [brand.md](../brand.md)。
2. **中文名（拟议）**：**步步登台**——「步步」对应 Step by step/阶段，「登台」对应舞台；纯中文四字满足 ICP 名称规范。前置条件：第 9/42/35 类商标检索通过。
3. **主 slogan（拟议）**：**AI 搭台，你登台**；长版副标：你拍板，AI 搭台；一步一步，登上舞台。
4. **设计红线**：门禁只设在主观决策点，宣传必须同步传达"该拍板才停，其余 AI 全速"，防止"分阶段"被误读为低效。

## 备选与否决理由

| 备选 | 否决理由 |
|---|---|
| 拾阶智造 | "拾阶"书面化，目标用户（非技术老板）理解成本高；制造感强、舞台感弱 |
| 进阶舞台 | "进阶"指向用户成长而非任务分阶段，双关偏移 |
| 智造舞台 | 缺 step-by-step 一半词源，通用感强 |
| 蜗牛登台/蜗牛工坊 | 呼应蜗牛 logo 但「蜗牛」在 9/42 类有强在先主体（蜗牛游戏等），商标冲突概率高 |
| 音译（如"思塔简特"） | 无语义承载，违背"人话"文案原则 |

## 后果

- 正：品牌名-中文名-slogan-产品机制四层同源（台/阶/步），叙事一致性强；备案名称与品牌统一
- 负：品牌强绑定"分阶段"哲学——若未来产品转向更高自动化（减少门禁），叙事需调整；缓解：红线条款已把"自动化程度提高"解释为"客观执行更快"，与"主观决策在人"不冲突

# 品牌叙事与品牌语系统

> 状态：**哲学已确认；中文名与主 slogan 拟议中（待商标检索）** ｜ 最后更新：2026-07-05 ｜ 决策记录：[ADR-008](adr/ADR-008-brand-naming.md)

## 一、名字的三重词源

**Stagent = Stage + Agent**，其中 Stage 承载三层含义，共同构成产品哲学：

| 词源 | 含义 | 对应产品事实 |
|---|---|---|
| Stage = **舞台** | 人类在 AI 辅助下释放想象力，最终**站上舞台展示成果** | 案例墙（P2）→ 作品广场（P4）；交付的应用本身就是客户的"登台" |
| Stage = **阶段**（in stages） | 复杂任务**分阶段推进**，每阶段结束是一个人类决策节点 | 五阶段流程 + 阶段门（gate）机制 |
| Step by step | **逐步**，而非一键黑盒 | 门禁双条件（字段齐全 + AI 判断就绪）才可拍板；可回退 |

## 二、核心价值主张：分阶段 = 人类的在场

> **"分阶段，不是效率的妥协，而是人类的在场。"**

这不是功能特性，是对**人机关系**的价值主张：

1. **分阶段 = 保留人类的决策节点**。一键生成时，人只是指令发出者和结果接收者，中间全是黑盒；分阶段时，每个阶段结束人类都有"喊停、调整、纠正"的权力。
2. **分阶段 = 降低失控风险**。一键生成出错只能推倒重来；分阶段出错回退到那一步修正。人握方向盘，AI 是导航仪。
3. **分阶段 = 想象力落地的方法论**。模糊想象 → 清晰目标 → 可执行步骤 → AI 执行 + 人实时调整 → 完整作品 → 登台。
4. **分阶段 = 归属感**。黑盒产品的成果是"AI 做的"；Stagent 的成果是"**我做的**（AI 帮我做的）"——这层归属感是 B2B 信任与传播的根基。
5. **分阶段 = 责任链**（对企业客户很重要）：需求是你确认的、验收是你勾的、上线是你批的——每个决策有主，出了问题知道在哪一环。

### 与同类产品的本质区别

| | 黑盒模式（一键生成） | Stagent（分阶段） |
|--|--|--|
| 输入 | 指令 | 目标 |
| 过程 | 黑盒出结果 | 每阶段人类参与 |
| 人类角色 | 旁观者 | 主导者 |
| 出错时 | 推倒重来 | 回退该阶段修正 |
| 成果归属 | "AI 做的" | "我做的" |

### ⚠️ 必须守住的设计红线（防止品牌被读成"慢、麻烦"）

"人类在场"≠"处处打断"。**门禁只设在主观决策点**（不可逆动作、价值判断），客观执行 AI 自动跑完——这正是 `gateReadiness` 双条件机制的设计本意。品牌宣传中必须同时讲清"该你拍板时才停下来，其余时间 AI 全速前进"，否则差异化会被误读为低效。

## 三、品牌故事（官网/介绍用）

> **Stagent = Stage + Agent**
>
> 在 AI 狂飙的时代，Stagent 选择了一条不同的路。
>
> 我们不追求"一键完成"，而是把复杂任务拆成一个个清晰的 Stage（阶段）。在每个阶段，你都是主导者——AI 负责执行，你负责判断、调整、决策。
>
> 因为我们认为：**AI 的真正价值，不是替代人类，而是让人类在每一个阶段都能更好地释放想象力。**
>
> 最终，你带着完整的成果，站上属于你的 Stage（舞台）。
